# Library Management System

## 🚀 Tech Stack
- Java
- Spring MVC
- JSP
- MySQL
- JDBC

## 📌 Features
- Add Book
- Add Student
- Issue Book
- View Issue Details

## ⚙️ Setup Instructions
1. Clone the repo
2. Import as Maven Project in Eclipse
3. Configure MySQL database
4. Run on Tomcat Server
